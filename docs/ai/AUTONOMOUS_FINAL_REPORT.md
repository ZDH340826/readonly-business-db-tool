内嵌 Java runtime 的标准 `runtime/conf/net.properties` 含回环地址 `127.0.0.1`，属于第三方运行时默认本机配置，不是现场或生产信息。项目自有配置、脚本、文档和预检文本的 IPv4/JDBC/凭据/绝对路径命中为 0；诊断 ZIP 使用更严格规则，任何 IPv4 命中均为 0。
