# Release v0.4.0

Date: 2026-07-04

## Summary

v0.4.0 closes the point status dashboard release for onsite shortage monitoring. Operators can now see each configured point as `有料`, `无料`, `未查到`, or `停用`, while group monitoring uses each group's own check interval and real elapsed time for alerts.

## Included

- Point status dashboard prioritized on the point shortage page.
- Per-point status cards for use and backup points.
- Per-group check interval configuration.
- Per-group alert duration configuration.
- Real elapsed-time alert evaluation.
- Chinese operator-facing group status and alert text.
- Default packaged sample group with `group.0.rule.backupThresholdParticipates=true`.

## Safety Boundary

- No SQL editor.
- No table editing UI.
- No INSERT, UPDATE, DELETE, DDL, or save-to-database workflow.
- Database access remains readonly.
- Passwords are entered at runtime and are not saved.
- Public package contains sample point identifiers only.
- Real IPs, usernames, passwords, production point groups, and real point lists must remain local.
