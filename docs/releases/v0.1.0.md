# Release v0.1.0

Date: 2026-06-30

## Summary

First public release of the readonly business database tool.

## Included

- Java Swing desktop application.
- Connection management for sample PostgreSQL and local H2 profiles.
- Readonly database browser for schema, table/view, column, and row preview inspection.
- Shelf point shortage alert page with per-point monitoring intervals.
- Local H2 test database tooling.
- Build script that compiles, tests, packages a runtime, and creates a versioned zip.

## Safety Boundary

- No SQL editor.
- No table editing UI.
- No write actions in the browser.
- PostgreSQL connections request readonly transactions.
- Public defaults use sample values only.
