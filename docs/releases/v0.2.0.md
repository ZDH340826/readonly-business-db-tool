# Release v0.2.0

Date: 2026-06-30

## Summary

Adds group-level shortage monitoring for operational areas and material groups.

## Included

- Point group configuration with area, group name, material name, one use point, and multiple backup points.
- Group alert rule configuration:
  - use point must be empty,
  - available backup count must be below threshold,
  - condition must remain true for the configured duration.
- Automatic checks every 60 seconds.
- Manual "check now" flow.
- Acknowledged alert state that suppresses repeat dialogs until recovery.
- CSV check and event logs under `logs/`.
- Local H2 sample database with one use point and four backup points.
- Obsidian-ready Chinese user manual.

## Safety Boundary

- No SQL editor.
- No table editing UI.
- No data mutation buttons.
- Group monitoring uses fixed SELECT queries only.
- Runtime logs are local files only.
- Packaged configs contain sample identifiers only.
