## C. 本地 H2 功能

- [ ] “连接管理”可选择并“测试连接”本地测试库。
- [ ] “监控总览”点击“立即检测”后显示公开样例点位。
- [ ] 八个导航页面均可进入且非空。
- [ ] “数据查询”可执行结构化只读查询、分页并“导出当前结果”。
- [ ] “数据源浏览器”可“刷新 Schema”“加载表/视图”“预览前100行”。
- [ ] “日志与系统”可“执行自检”和“导出诊断包”。
- [ ] 诊断 ZIP 只含六个允许文件，内容已脱敏。

## D. 报警与异常语义

- [ ] 正常、观察中、报警、已关注、已恢复均显示中文文字和语义颜色。
- [ ] 查询失败显示“查询失败，数据不可用”，不显示为缺料、正常或已恢复。
- [ ] 查询失败期间不触发业务缺料弹窗；恢复后缺料持续时间从零重新计算。
- [ ] “标记已关注”后同次短缺不重复弹窗，真实恢复后状态清理。
- [ ] “停止监控”或切换连接后，旧任务不覆盖新状态。
